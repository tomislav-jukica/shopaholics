# Shopaholics App

This project includes a .NET backend and an Angular frontend. Follow these steps to run the app locally.
You can use an already created user or you can registrate your own.

Login information:

---

## Prerequisites

- [.NET 10 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)
- [Node.js](https://nodejs.org/) (includes npm)
- Angular CLI (optional, can be installed with `npm install -g @angular/cli`)

---

## Backend (API)

1. Open a terminal and navigate to the backend project folder:
   cd Shopaholics.Server
2. Run the backend:
   dotnet run

## Frontend (Angular)
1. Open a terminal and navigate to the frontend project folder:
   cd Shopaholics.client
2. Install dependencies:
   npm install
3. Start the frontend server:
   ng serve
